# Rename Image Empties

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

**Аддон для Blender, который быстро переименовывает выбранные пустышки с изображениями в соответствии с именами их файлов.**


## Немного предыстории

Я (как и многие другие 3D-художники) часто использую референсы прямо в 3D Viewport, но когда изображений слишком много, весь проект заполнен сплошными `Пустышками.XXX`. Меня всегда слегка раздражало, что нет функции, которая называла бы пустышки в соответствии с именами файлов изображений, так что… я решил сделать свой собственный аддон/расширение. Результат мне так понравился, что я решил поделиться им с вами, коллеги-художники. Не стесняйтесь делиться мыслями, мнениями и фидбеком (не «ты долбой, чё за хню ты сделал», а нормальные отзывы и указания на то, что исправить/улучшить).

## Возможности

- Переименовывание нескольких (всех, если нужно) пустышек с изображениями за один раз
- Опция сохранить или убрать расширения файлов (`.png`, `.jpg` и т.д.)
- Полная отмена действий (переименования сохраняются в истории проекта, как и любые другие действия)

## Установка

1. Скачайте последний `.zip` из раздела [Releases](https://github.com/ShadowGaszin/blender_rename_image_empties/releases)
2. Откройте Blender → **Edit** → **Preferences** → **Extensions** <sub>(4.1+)</sub>/**Add-ons** <sub>(2.80-4.1)</sub>
3. Нажмите **Install** и выберите `.zip` файл
4. Отметьте галочкой аддон

## Использование

### Через боковую панель (N-panel)
1. Выберите одну или несколько пустышек с изображениями в 3D-сцене
2. Откройте боковую панель клавишей `N`
3. Перейдите на вкладку **Image Empties**
4. Включите/выключите "Keep Extension" (Сохранить расширение) при желании
5. Нажмите **Rename Image Empties**

### Через меню Object
1. Выберите ваши пустышки с изображениями
2. Перейдите в **Object** → нажмите **Rename Image Empties**
3. Если нужно, выберите опцию "Keep Extension" (Сохранить расширение) в меню внизу
